Preguntas frecuentes

¿Qué es lo que hace nuestro sistema de información?

Nuestro sistema de información, funciona escaneando el código que tiene pegado cada producto en nuestro supermercado para luego sacar la información de, por ejemplo, el precio, de nuestra base de datos, en la cual se almacena toda la información de los productos que vendemos.

¿Como se hace?

Primero se introduce la información del producto que llega en la base de datos, para luego ponerle un identificador para el escáner mediante un código que se pega en el producto físico.

¿Con que frecuencia se presenta?

Se presenta con bastante frecuencia, ya que al ser un supermercado se venden bastantes productos por día.

¿Qué tan grande es el volumen de transacciones o decisiones?

No es tan grande, debido a que los datos de los productos que se sacan de la base de datos son pequeños.

¿Cual es el grado de eficiencia con el que se efectúan las tareas?

Muy rápido, el escaner tarda muy poco en sacar los datos de la base de datos.

¿Existe algún problema? ¿Que tan serio es? ¿Cual es la causa que lo origina?

El problema más notable es el tamaño creciente de la base de datos, ya que mientras mas productos hay, mas datos se meten y por lo tanto la base de datos se hace más grande, ocupando más espacio.
